# 戦国武将当てゲーム

戦国武将のヒントを見ながら、誰なのかを当てるブラウザゲームです。

## 概要

このゲームは、表示されるヒントをもとに戦国武将の名前を当てるクイズゲームです。正解数や連続正解によってスコアが変動し、ミスが2回になるとゲームオーバーになります。さらにオンラインランキング機能もあり、他のプレイヤーとスコアを競うことができます。

## 主な機能

- ニックネーム登録
- 30秒制限タイマー
- ヒント表示機能
- スコアシステム
- 連続正解ボーナス
- ミスカウントによるゲームオーバー
- 効果音（正解・不正解）
- オンラインランキング機能

## スコアルール

- 正解：+20点
- 2連続正解以降：追加+10点
- 不正解：-5点
- 時間切れ：-5点
- ミス2回でゲーム終了

## 遊び方

1. ニックネームを入力する
2. STARTボタンを押す
3. ヒントを見ながら武将を予想する
4. 答えを入力して送信する
5. スコアを伸ばしてランキング上位を目指す

## 制作について

このプロジェクトは、フロントエンドとバックエンドの両方を含むWebアプリケーションとして制作しました。

単なるクイズではなく、サーバーを介したデータ保存やランキング機能を実装することで、実際のWebサービスに近い構造を学習する目的もあります。

また、ゲームとしての面白さと学習要素の両立を意識して設計しています。
